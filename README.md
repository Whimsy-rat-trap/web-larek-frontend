# Проектная работа "Веб-ларек"

# Описание структуры проекта
Проект представляет собой интернет-магазин с следующими ключевыми компонентами:

# Основные страницы
1.	Главная страница - центральная точка входа в приложение
2.	Модальное окно корзины - отображает товары положенные в корзину
3.	Модальное окно формы оплаты - сбор платежной информации
4.	Модальное окно контактных данных - сбор информации о доставке
5.	Модальное окно товара - детальная информация о продукте

# Компоненты и их функционал
1. Главная страница
- Назначение: Отображение каталога товаров
- Функционал:
	- Получает ProductListResponse с сервера
	- Отображает массив ProductCard[]
2. Карточка товара
- Назначение: Отображение каталога товаров
- Содержит данные:
	- title - название товара
	- price - цена
	- category – категория
	- image - изображение товара
- Функционал:
	- Открывает модальное окно с деталями товара
	- Позволяет добавить товар в корзину
3. Корзина
- Назначение: Хранение товаров для покупки
- Содержит данные:
	- items - список товаров
	- total - общая стоимость
- Функционал:
	- Открывает форму оплаты
	- Позволяет удалять товары из корзины
4. Модальные окна
4.1 Окно товара
- Назначение: Отображает описание продукта
- Отображает:
	- Все поля карточки товара
	- Подробное описание товара
4.2 Окно корзины
- Назначение: Отображение и управление содержимым корзины
- Отображает:
	- Список выбранных товаров
	- Итоговую сумму
- Функционал:
	- Удаление товаров
	- Переход к оформлению заказа
4.3 Форма оплаты
- Собирает:
	- Платежные данные
	- Способ оплаты
4.4 Форма контактов
- Собирает:
	- Адрес доставки
	- Контактный email
	- Телефон для связи
4.5 Форма успешной покупки
- Назначение: Ответ API при успешном оформления заказа
- Отображает:
	- Изображение галочка
	- Итоговую сумму

# Поток данных
1.	Пользователь просматривает товары на главной странице
2.	При клике на карточку открывается детальная информация о товаре
3.	Добавление товара в корзину обновляет содержимое корзины
4.	Корзина рассчитывает общую сумму
5.	Оформление заказа объединяет данные:
	- Товары из корзины
	- Платежные данные
	- Контактную информацию

# Взаимодействие компонентов
1.	Главная страница -> Карточки товаров -> Модальное окно товара
2.	Главная страница -> Иконка корзины -> Модальное окно корзины
3.	Корзина -> Форма оплаты -> Форма контактов -> Подтверждение заказа


Стек: HTML, SCSS, TS, Webpack

Структура проекта:
- src/ — исходные файлы проекта
- src/components/ — папка с JS компонентами
- src/components/base/ — папка с базовым кодом

Важные файлы:
- src/pages/index.html — HTML-файл главной страницы
- src/types/index.ts — файл с типами
- src/index.ts — точка входа приложения
- src/scss/styles.scss — корневой файл стилей
- src/utils/c-nstants.ts — файл с константами
- src/utils/utils.ts — файл с утилитами

## Установка и запуск
Для установки и запуска проекта необходимо выполнить команды

```
npm install
npm run start
```

или

```
yarn
yarn start
```

Для корректной работы проекта требуется создать файл .env в корневой директории:

```
touch .env
```

Внутри .env так же необходимо указать базовый URL API-сервера:

API_ORIGIN=https://larek-api.nomoreparties.co

## Сборка

```
npm run build
```

или

```
yarn build
```
